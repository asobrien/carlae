Carlae

Carlae is a private URL shortener that is designed to work with your domain.

Carlae includes a simple login management and registration system. User registration is built around a web-of-trust model, where existing users can invite new users to the service.

Getting Started

Of course, you'll need your own domain and a server to host this. After that the requirements are pretty straightforward.

Features

The features of carlae are purposely limited. Form is built around function. With that said there are a few features built in.
